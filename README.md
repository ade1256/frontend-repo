# Frontend Repo

### How to run this code ?
1. you should run backend repo first https://github.com/ade1256/backend-repo
2. clone this repository 
3. run `npm install`
4. run `npm run dev`

### Url and ports
1. homepage `http://localhost:3000`
2. login page `http://localhost:3000/login`
